# Dropshipzone 应用仓库

本仓库包含 Dropshipzone 上架工具的最新版本文件。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `Dropshipzone全品类上架模板_V7.4.html` | V7.4 最新版网页 |
| `app.html` | Electron 软件界面 |
| `main.js` | Electron 主进程 |
| `package.json` | 软件包配置 |

## 🚀 使用方式

1. 点击页面上方的 **Code** 按钮 → **Download ZIP**
2. 解压 ZIP 文件
3. 使用 `Dropshipzone全品类上架模板_V7.4.html` 或 Electron 软件

**Q: V7.1 翻译功能不显示？**
A: 重启应用后，点击右上角的 🌐 按钮。

**Q: 如何反馈问题？**
A: 联系开发者或在 GitHub 创建 Issue。
