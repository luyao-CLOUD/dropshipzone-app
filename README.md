# Dropshipzone 应用仓库

本仓库包含 Dropshipzone 服装上架工具的最新版本文件。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `Dropshipzone全品类上架模板_v7.1.html` | V7.1 网页版（推荐） |
| `Dropshipzone服装鞋类上架工具_V1.3.html` | V1.3 服装鞋类专用版 |
| `version.json` | 版本信息 |
| `Dropshipzone_自动更新.bat` | 自动更新脚本（Windows） |
| `Dropshipzone_GitHub上传.bat` | 上传脚本（开发用） |

## 🔄 如何更新

### 方法一：自动更新（推荐）

1. 下载本仓库的 `Dropshipzone_自动更新.bat`
2. 双击运行
3. 脚本会自动从 GitHub 下载最新版本并安装

**要求：**
- Windows 10 或更高版本
- 网络连接正常

### 方法二：手动更新

1. 点击页面上方的 **Code** 按钮 → **Download ZIP**
2. 解压 ZIP 文件
3. 复制需要的 HTML 文件到桌面或安装目录

**Q: V7.1 翻译功能不显示？**
